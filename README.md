## 🗺️ The Experience

Instead of a boring scroll page, this portfolio is a **treasure map adventure**. A rolled parchment scroll sits on an aged map — unroll it to reveal a hand-drawn island with four destinations, each a chapter of my story.

```
🪄  Scroll Cover
      ↓  click to unroll
🗺️  Treasure Map Island
      ↓  click a destination
☁️  Cloud Transition  ←  3D model loads silently in background
      ↓  clouds part
📖  Chapter
      ↓  ← Map button
🗺️  Back to Map
```

## ✨ Highlights

- 🪄 &nbsp;**Parchment scroll** — horizontal unroll animation with 3D wooden cylinder rolls
- 🗺️ &nbsp;**Hand-drawn SVG map** — island blob, palm trees, volcano, ship, compass rose, dashed treasure paths
- ☁️ &nbsp;**Cloud transition** — cinematic sky scene masks all 3D loading lag completely
- 🤖 &nbsp;**Waving 3D avatar** — GLB model, plays greeting animation, speech bubble, tech tags
- 🖥️ &nbsp;**Windows 7 simulation** — working desktop with draggable windows & real apps
- 🏟️ &nbsp;**3D Stadium** — Old Trafford model orbiting with polaroid memories
- 📺 &nbsp;**Video reel** — iPad UI, hero video, scrollable movie & show rows
- 🔊 &nbsp;**Ambient audio** — treasure sound on cover, stops on interaction
- ⚡ &nbsp;**Zero lag navigation** — chapters stay mounted in memory, shown/hidden with opacity
